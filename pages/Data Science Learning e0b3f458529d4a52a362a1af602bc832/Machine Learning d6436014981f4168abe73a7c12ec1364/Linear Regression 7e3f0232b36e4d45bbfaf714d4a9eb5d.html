Assumptions of Linear Regression

R-squared (Coefficient of Determination)

(Video from Khan Academy)

Question: What % of the variation in y is described by the variation in x?

Alternate question: If we fit a line on the data points, how good is the fit of that line?

  • Total variation in y is the distance of each value of y from the mean of y. It is also called SST.
  • What % of the total variation is NOT explained by the regression line?

    Numerator will be SSE since this the error which is not explained by the regression line. If the regression line doesn’t fit the data well, then residuals will be high and so SSE would be high.

  • What % of the variation in y is described by the variation in x? OR what % of variation in y is described by the regression line

What is the shortcoming of R2R^2R2﻿

  1. # of Independent variables: It increases as number of independent variables increase.
  1. Sample Size: It tends to increase with the sample size which makes it challenging to compare R2 values across datasets with different sample sizes
  1. Assumes Linearity: R2 is most meaningful where the relationship between the independent and dependent variables is assumed to be linear
  1. No Information on Prediction Accuracy: R2 does not provide information about the accuracy of predictions for new, unseen data. It measures the goodness of fit for the observed data but does not assess the model's generalisation performance.

Measures for Linear Regression

Feature Importance

  1. Coefficients Magnitude: Magnitude of the coefficient indicates the strength and direction of the relationship between feature and predicted outcome. Larger magnitudes imply a more significant impact. Feature scaling should be done for using this.
  1. p-values: Lower p-values suggest a more significant relationship between the feature and the target variable. However, be cautious with using p-values
  1. Stepwise Selection: Iteratively add or remove features based on their impact on the model's performance

Regularisation

L1 = Lasso

L2 = Ridge

How regularisation works:

  • Naive way to reduce loss would be make the coefficients (β\betaβ﻿) to very low values
  • However, that increases loss from squared errors as model doesn’t fit
  • Best way would be to set beta high enough for features which add value by keeping squared errors low and set beta to be very low (or zero) for features which don’t add any value

Feature selection: L1 regularisation helps in feature selection, since it shrinks coefficient of less important features to zero

Feature Normalisation

Why is this needed?

  1. impact of one feature on the regression model may dominate over others, if not normalised
  1. accelerate the convergence of optimisation algorithms like gradient descent
  1. makes it easier to interpret the coefficients of the regression model
  1. regularisation methods penalise large coefficients. If features are on different scales, the regularisation term may unfairly penalise some features more than others

Methods

  1. Min-max normalisation
  1. Z-score normalisation
