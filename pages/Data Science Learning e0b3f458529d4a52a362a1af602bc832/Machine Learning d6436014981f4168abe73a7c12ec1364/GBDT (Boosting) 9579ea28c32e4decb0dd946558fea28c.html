Regression

  1. Initial prediction: Avg. value
  1. Build trees iteratively to predict the residuals (actual-predicted) from the previous tree
    1. Residual is the gradient of loss function w.r.t. predicted. This give the name gradient descent to this algo
    1. Remember the gradient descent formula for regression, we are doing kind of the similar thing:

      New Prediction = Old Prediction - Alpha x Gradient w.r.t prediction

  1. Combine the output from trees using a learning rate

Loss function is minimised for initial value = avg. value

  • because if we solve δLossδPredicted=0\frac{\delta Loss}{\delta Predicted} = 0δPredictedδLoss​=0﻿, we get Predicted = Actual

This is the same reason for value in the leaf node being average of all values

Classification

  1. Initial prediction: log(odds)
  1. Build trees to predict residuals
  1. Leaf node would have log(odds) prediction

Light GBM

  • Find the best leaf node at each split
  • Results in better performance
